Before you begin

This is an Android class which will require a Xamarin.Android development environment to complete the coding exercises. Please read the setup requirements if you need help installing the Xamarin development environment.

In addition, you will need to install the optional Google Play services SDK library from the Android SDK Manager and get an Android emulator with Google Play services installed, or (preferably) a device to run the exercise on.

How do I setup Google Play services?

Testing with Google's pre-built Android image

The easiest method to make sure Google Play services is available is to download a Google emulator image with the appropriate APIs already available (remember to use x86 for performance). Using the Android SDK Manager (Tools > SDK Manager in Visual Studio for Mac, Tools > Android > Android SDK Manager... on Windows), download one of the "Google APIs" or "Google Play" emulator images.

With the image downloaded, create a new emulator using that image with the Android Emulator Manager (Tools > Google Emulator Manager in Visual Studio for Mac, Tools > Android > Android Emulator Manager in Visual Studio on Windows).

Now when you deploy to an emulator, choose the emulator with Google APIs loaded, and Google Maps APIs will work in your emulator. If you choose an emulator or device without the Play services APIs, it will display an error message.

Overview

In this course, we will continue our discussion of using Google Maps in Android applications by adding in the ability to work with the current location and search for points of interest around us. We will then annotate the map with markers and shapes to draw special features related to the things in our area.

It is assumed that you have already taken the AND230 - Google Maps class prior to attending this one. If not, you should register for that class first as it has a lot of prerequisite knowledge.

Objectives

  1. Determine the current location
  2. Search for points of interest around the device

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Setup the Google Maps API key
    Exercise
    In this group exercise, we will add your Google API key to the starting application and ensure that the mapping support is all working.
  2. Working with the Location Manager
    Exercise
    This exercise will explore the LocationManager service and how to identify your current location and watch it change.
  3. Identifying a provider based on criteria
    Exercise
    Here we will modify our location registration code to ask Android which provider we should use by providing some crtieria.
  4. Adding support for Location Services
    Exercise
    Next, in this group exercise we will look at integrating the LocationServices support included with Google Play and how to fall back on LocationManager when it's not available.
  5. Using Geocoder to perform reverse geocoding
    Exercise
    In this exercise you will look up a street address from a specific coordinate on our map using the Geocoder class.
  6. Using Geocoder to find a specific address
    Exercise
    In this final exercise you will find a known address on the map using the Geocoder class.
